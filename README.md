# To-Do List Application

A modern, responsive to-do list web application with user authentication and local storage.

## Features

- ✅ User Authentication (Login/Signup)
- ✅ User-specific task management
- ✅ Add tasks with dates
- ✅ Check/uncheck tasks
- ✅ Tasks persist in browser storage
- ✅ Modern, responsive design
- ✅ Smooth animations and transitions

## How to Use

1. **Getting Started**
   - Open `login.html` in your browser
   - Create an account by clicking "Sign up"
   - Login with your credentials

2. **Adding Tasks**
   - Enter a task in the text input
   - Optionally select a date
   - Click "Add" button

3. **Managing Tasks**
   - Check the checkbox to mark as complete
   - Tasks are automatically saved
   - Each user has their own task list

4. **Logout**
   - Click the "Logout" button in the top-right corner
